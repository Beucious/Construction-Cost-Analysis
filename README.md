# Lagos Construction Cost Analysis 2024

A cost variance analysis dashboard built in Excel (Google Sheets) across 17 active construction and infrastructure projects in Lagos, Nigeria.


---

## Project Summary

This project analyses budgeted vs actual construction costs across 51 line items spanning Labour, Materials, and Equipment categories. The goal was to identify cost overruns, spot spending trends, and present findings in a clean, decision-ready dashboard.

---

## Key Findings

- **Total Budget:** ₦165.1M across 17 projects
- **Total Actual Spend:** ₦174.5M
- **Total Variance:** ₦9.4M over budget
- **Materials** was the highest overspend category
- **Victoria Island Office Complex** and **Oshodi Transort Interchange** recorded the largest cost overruns
- Spending peaked in **September and December 2024**
